# 通海港全道岔支线中心线候选说明

本轮目标是把人工反馈过的道岔支线统一整理成一个 QGIS 验收层，并在交付前用原始 DOM 全分辨率裁图做自检。旧的模板层、渡线层和 LAS 验证层都保留；本轮只新增汇总层，不覆盖前序产物。

## 合并规则

`TA01/TA02` 与 `TA04/TA05` 不再作为四条独立模板支线处理，而是采用成对渡线逻辑：

- `CX01 = TA02 -> TA01`，采用 `turnout_crossover_las_endpoint_locked_centerlines`。
- `CX02 = TA05 -> TA04`，采用 `turnout_crossover_las_endpoint_locked_centerlines`。

剩余独立道岔支线继续采用前序模板/特殊形态候选：

- `TA03`
- `TA06`
- `TA07`
- `TA08`
- `TA09`

因此主验收层一共 7 条线：`CX01`、`CX02`、`TA03`、`TA06`、`TA07`、`TA08`、`TA09`。

## QGIS 验收建议

优先加载 `all_turnout_branch_centerlines.shp`，然后加载同目录的 `.qml` 样式。属性表里重点看：

- `branch_id`：本轮统一编号。
- `src_type`：`crossover_las_endpoint_locked` 或 `remaining_template_or_special`。
- `qa_status`：我的原图裁切自检状态。
- `self_note`：逐条线的自检说明。

如果只想先找风险点，优先看 `TA08`。如果要验证成对渡线改法是否真的解决起终点问题，重点看 `CX01` 和 `CX02` 的起终点。
